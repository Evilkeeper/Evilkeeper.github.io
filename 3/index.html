Сайты GitHub Pages должны воздерживаться от

  • нелегального контента и контента запрещённого этой или этой фигнёй, которую никто не читает
  • жестокого и угрожающего контента или деятельности
  • чрезмерной автоматизированной деятельности (как, например, спам)
  • деятельности, которая ставит под угрозу пользователей GitHub или GitHub services
  • схем быстрого заработка
  • контента непристойного характера
  • контента, искажающего вашу личность или цель сайта
